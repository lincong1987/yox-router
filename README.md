# yox-router

## Install

NPM

```shell
npm install yox
npm install yox-router
```

```js
import Yox from 'yox'
import YoxRouter from 'yox-router'
Yox.use(YoxRouter)
```

CDN

```html
<script src="https://unpkg.com/yox@latest"></script>
<script src="https://unpkg.com/yox-router@latest"></script>
```
